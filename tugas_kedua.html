Tugas Kedua - Pengetahuan Umum

1. Apa yang kamu ketahui tentang Version Control (usahakan pergunakan wawasan & bahasa sendir)

  • Version Control Syetem

    Sistem yang mencatat setiap perubahan terhadap sebuah berkas atau kumpulan berkas sehingga pada suatu saat anda dapat kembali kepada salah satu versi dari berkas tersebut

2. Jelaskan apa itu GIT

  • GIT

    Sebuah version control system yang digunakan para developer untuk mengembangkan software secara bersama-bersama. Dan bukanlah sebuah bahasa seperti halnya HTML,CSS atau Js bukan pula sebuah konsep atau aturan baku dalam pemrograman.

3. Apa yang kamu ketahui tentang Framework khususnya Web Framework, sebutkan apa saja web frameworkyang kamu ketahui

  • Web Framework

    Sebuah kerangka kerja untuk memudahkan para programer untuk membuat sebuah aplikasi web yang didalamnya ada berbagai fungsi diantaranya plugin, dan konsep untuk membentuk suatu sistem tertentu agar tersusun dan tersetruktur dengan rapih.

    Macam-macam Web Framework:

    Laravel

    Yii

    Code Igniter

    Symfoni

    Zend Framework

    Cake PHP

    Fuel PHP

    DLL

4. Jelaskan istilah - istilah berikut: REST API, SQL, HTTP, UI Framework

  • REST API

    REST (Representional State Transfer) adalah suatu arsitektur metode komunikasi yang menggunakan protokol HTTP untuk pertukaran data dan metode ini sering diterapkan dalam pengembangan aplikasi. Dimana tujuannya adalah untuk menjadikan sistem yang memiliki performa yang baik, cepat dan mudah untuk di kembangkan (scale) terutama dalam pertukaran dan komunikasi data.

  • SQL

    SQL merupakan bahasa atau bisa juga disebut sebagai kumpulan perintah standar yang biasa digunakan untuk berkomunikasi dengan database.

  • HTTP

    HTTP adalah sekumpulan aturan protokol aplikasi yang berjalan di atas rangkaian protokol TCP / IP (Internet) untuk mentransfer file-file seperti text, gambar, suara, video, dan file lainnya.

  • UI Framework

    Sebuah kerangka kerja untuk memudahan membangun user interface pada pekerjaan front-end programmer, yang bisanya untuk memudahkan penataan disain untuk sebuah user interface.

5. Apa yang dimaksud dengan RDBMS dan noSQL, jelaskan perbedaannya

  • RDBMS (Relational Database Management System)

    DRBMS adalah sebuah tempat/lokasi atau sekumpulan berkas data, yang telah terkomputerisasi yang bertujuan untuk memelihara informasi. Serta memuat informasi tersebut(Relational), khususnya jika informasi itu memang sedang dibutuhkan.


    noSQL

    Sebuah berkas data yang tidak seperti sekumpulan data yang menggunakan tabel dalam penyusunan datanya, nosql menggabungkan semua database tidak membedakan jenisnya dan tanpa karakteristik umum. Tapi nosql database ini memiliki kecepatan yang super cepat dibanding dengan berkas data biasanya, pencariannya lebih terfokus. Nosql sebetulnya tidak sepenuhnya menyimpan data dengan cara tidak terstruktur, terkadang ada miripnya dengan berkas data biasanya dengan sedikit susunan pada saat tertentu bila dibutuhkan.

  • Perbedaan RDBMS dengan noSQL

    RDMS merupakan sekumpulan berkas data yang berstuktur dan relasional, sedangkan noSQL merupakan sekumpulan berkas data yang tidak tersetruktur dan tidak relasional.
