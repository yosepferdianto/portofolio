Simple Parser

Buatlah sebuah fungsi dengan nama simple_parser untuk me’replace kata di dalam sebuah text (sebagai input pertama) dengan dengan kata lain (sebagai input kedua denga tipe key value).

misal :

INPUT PERTAMA: `Nama saya {nama}, usia {usia} tahun, saya berprofesi sebagai {profesi}`

INPUT KEDUA: { nama: ‘Tri Pamungkas’, usia: 30, profesi: ‘database developer’}

OUTPUT: `Nama saya Tri Pamungkas, usia 30 tahun, saya berprofesi sebagai database developer`

function simple_parse(nama, usia, profesi) {
	var json = '{"nama": "' + nama + '", "usia": ' + usia + ', "profesi": "' + profesi + '"}';

	// Converting ke JSON object to JS object
	var obj = JSON.parse(json);

	// Melihat data yang dihasilkan obj melalui console
	console.log('Input Parse: ' + json);
	console.log('Nama saya ' + obj.nama + ' usia ' + obj.usia + ' tahun, saya berprofesi sebagai' + obj.profesi);

	return 'Nama saya ' + obj.nama + ' usia ' + obj.usia + ' tahun, saya berprofesi sebagai ' + obj.profesi;
}
